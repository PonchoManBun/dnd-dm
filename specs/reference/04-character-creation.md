# 04 — Character Creation

## Overview

Character creation follows D&D 5e rules. The player builds a single character — no party, no companions by default (though summoning spells can create temporary allies). The AI DM guides the process conversationally, suggesting options and generating a backstory.

## Flow

1. **Choose Race** — Human, Elf, Dwarf, Halfling, Half-Orc, Gnome. Each grants racial traits per SRD (ability bonuses, darkvision, proficiencies, etc.).
2. **Choose Class** — Fighter, Wizard, Rogue, Cleric, Ranger, Paladin. Determines hit dice, proficiencies, starting equipment, and class features.
3. **Roll Stats** — STR, DEX, CON, INT, WIS, CHA. Standard 4d6-drop-lowest method. Player assigns scores to abilities.
4. **Choose Alignment** — The full D&D 9-point grid (Lawful Good through Chaotic Evil). Affects NPC reactions and DM narrative framing.
5. **Choose Skills** — Class-determined skill proficiency picks from the full D&D skill list.
6. **DM Generates Backstory** — Based on race, class, and alignment, the AI DM writes a short backstory. No predefined background templates — each backstory is unique.
7. **Name & Confirm** — Player names the character, reviews the sheet, and enters the tavern.

## Starting Conditions

- **Level 1**, 0 XP
- **Starting equipment** per class (SRD defaults)
- **Starting gold** per class roll
- **No map** — must find or buy one
- **Languages** — racial defaults; DM handles additional language use naturally

## DM Role in Creation

The DM doesn't just present menus — it narrates the creation as if meeting the character for the first time. "A half-orc walks into The Welcome Wench. She looks like she's killed before, and not always on purpose. What's her name?"
